# 🛒 Sistem Kasir Swalayan (Web App)

Aplikasi kasir berbasis *web* yang dibangun menggunakan Python dan Streamlit. Proyek ini mendemonstrasikan implementasi algoritma logika percabangan untuk diskon bertingkat dan perulangan bersyarat untuk memisahkan pecahan uang kembalian.

## 🚀 Fitur Utama
* **Kalkulasi Diskon Otomatis:** Menghitung diskon berdasarkan tier pembelanjaan secara *real-time*.
* **Pemisahan Uang Kembalian:** Mengurai total kembalian ke dalam pecahan rupiah terkecil yang tersedia secara efisien menggunakan operator *Modulo* dan *Floor Division*.
* **UI/UX Modern:** Antarmuka responsif yang sangat mudah digunakan.

## 🛠️ Teknologi yang Digunakan
* **Bahasa:** Python 3
* **Framework Web:** Streamlit

## 💻 Cara Menjalankan Aplikasi di Komputer Lokal
1. Pastikan Python sudah terpasang.
2. Install library yang dibutuhkan: `pip install -r requirements.txt`
3. Jalankan aplikasi: `streamlit run app.py`
